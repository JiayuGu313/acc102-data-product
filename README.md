## 1. Project Overview
This project develops a Python-based financial analysis tool to automatically query, analyse, and visualise listed company financial performance, cash flow lifecycle, and stock returns using WRDS data. It provides clear insights for fundamental analysis and business lifecycle evaluation.

## 2. Problem Definition & Target User
- **Analytical Purpose**: Query, analyse and visualise listed company financial performance, cash flow lifecycle, stock returns and key metrics based on WRDS data.
- **Target User**: Accounting/finance students, junior financial analysts, investment enthusiasts.
- **Business Context**: Support fundamental analysis, company lifecycle assessment and stock performance evaluation.

## 3. Data Source & Access Information
- **Database**: WRDS (Wharton Research Data Services)
  - Compustat (funda): Annual financial data
  - CRSP (msf / msfhdr): Monthly stock returns and prices
- **Data Access Date**: 2026-04-17
- **Variables**: Revenue, assets, net income, operating/investing/financing cash flow, monthly returns, prices, volume
- **License**: For educational use only via XJTLU WRDS account

## 4. Methods & Functions
1. Connect to WRDS and extract financial and stock data
2. Clean and process annual and monthly data
3. Classify company lifecycle using cash flow patterns
4. Calculate key performance metrics (return, volatility, Sharpe ratio, drawdown)
5. Visualise financial trends and stock performance
6. Export clean tables and reports to Excel
